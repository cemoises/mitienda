# MiTienda - E-Commerce Headless

Proyecto de comercio electrónico bajo el modelo Dropshipping con arquitectura moderna (Next.js + Tailwind CSS + Vercel).

## Estructura del Proyecto

- **`app/`** — Rutas y páginas de la aplicación (App Router de Next.js), incluyendo layouts, páginas y estilos globales.
- **`public/`** — Archivos estáticos (imágenes, íconos, favicon) servidos directamente.
- **`components/`** *(a crear)* — Componentes de UI reutilizables.
- **`lib/`** *(a crear)* — Lógica de negocio, integraciones y utilidades.

## Stack Tecnológico

- **Frontend:** Next.js, Tailwind CSS
- **Hosting:** Vercel
- **Control de Versiones:** GitHub
